LDA(线性判别分析)

​ 在Logistic回归中，通常是通过构建$Pr(Y = k |X = x )$的logistic函数。为此考虑另外一个可行的方法，我们通过分离地对$X$的每一类（比如说$Y$值确定）建立分布模型，并使用贝叶斯定理来估计$Pr(Y = k |X = x )$。为何要用另一种方法呢？原因如下：

  • 当每一类都足够分离开的时候，对logistic回归模型的参数估计就会变得不稳定，而LDA不会有这个问题。

  • 如果$n$足够小，并且$X$在每一个类别中都可以近似为正态分布，那么LDA会比logistics回归稳定。

  • LDA更擅长区分两类以上。

  • 首先，在实际情况中，train error rates通常比test error rates低，且$p$相对于$n$越高，越容易导致overfit.

  • 其次，一般的无效分类器将获得仅比LDA训练集错误率稍高的错误率。另外，LDA在辨别default的问题上具有较低的灵敏度。其原因是，LDA仅仅是尝试接近贝叶斯分类器，而贝叶斯分类器则具有最低的总的error rate，也就是说贝叶斯分类器会产生最小的分错类的点。

值得注意的是，LDA和QDA最大的区别是所有类别的协方差矩阵是否相等。这个原因是bias-variance trade-off。当存在$p$个预测子时，LDA的协方差矩阵总共有$p(p + 1) / 2$个不同变量，而QDA的协方差矩阵则有$Kp(p + 1) / 2$个变量。当假设$K$类具有相同的协方差矩阵时，LDA又变成了$x$的线性函数，也就是说只需要预测$Kp$个变量。因此，总体来说，LDA并不比QDA更灵活，因此其具有low-variance的特性。然而有个trade-off的点是，LDA容易遭受高的bias。粗略地说，LDA在相对更少的训练样本的情况下会表现得更好，而QDA则适合大样本的训练。
